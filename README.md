## 🌟 **프로젝트 개요**
자연친화적인 컨셉을 기반으로 순수하고 여행 감성을 담은 화장품 웹사이트 리뉴얼 프로젝트입니다.  
### **사용 기술**
- HTML, CSS, JavaScript, jQuery
- **Fonts**:

  - 🖋 **경기청년바탕**: 서브 폰트
  - ✨ **Continuous**: 디자인적 강조 요소
